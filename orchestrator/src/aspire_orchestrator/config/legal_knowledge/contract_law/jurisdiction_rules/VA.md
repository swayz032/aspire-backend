# Virginia Contract Law — Key Variations

## E-Signature Rules
Virginia adopted UETA (Virginia Code 59.1-479 through 59.1-497, effective 2000). Electronic signatures have full legal equivalence.

**Specific rules:**
- Virginia was among the first states to enact e-signature legislation
- Virginia Electronic Commerce Act supplements UETA
- Remote online notarization (RON) permitted under Virginia Code 47.1-2 (Virginia was the FIRST state to permanently authorize RON in 2012)
- Virginia has the most mature RON framework in the US

## Non-Compete Agreements
Virginia enacted restrictions on non-competes for low-wage workers under Virginia Code 40.1-28.7:1 (effective July 2020).

**Key restrictions:**
- Non-competes VOID for "low-wage employees" — defined as earning less than the state average weekly wage (approximately $1,343/week or ~$70,000/year as of 2025)
- Does not affect non-competes for employees earning above the threshold
- Does not affect non-solicitation or confidentiality agreements

**For non-restricted employees:**
- Virginia courts apply traditional reasonableness analysis
- Must protect legitimate business interest
- Must be reasonable in function (type of activity), geographic scope, and duration
- Courts will NOT blue-pencil — if any part is unreasonable, the entire covenant is void (strict approach)
- Consideration: Employment is sufficient for new hires; additional consideration may be needed for existing employees

## Statute of Limitations for Contract Disputes
- **Written contracts:** 5 years (Virginia Code 8.01-246(2))
- **Oral contracts:** 3 years (Virginia Code 8.01-246(4))
- **Breach of warranty (goods):** 4 years (UCC)
- **Fraud:** 2 years from discovery (Virginia Code 8.01-243)
- **Professional malpractice (legal):** 5 years (Virginia Code 8.01-246(2) — same as written contract)

## Notice Requirements
- **Residential lease termination (month-to-month):** 30 days (Virginia Code 55.1-1253)
- **Eviction (non-payment):** 5-day pay or quit notice (Virginia Code 55.1-1245)
- **Lease violation (material):** 30-day notice with 21-day cure period (Virginia Code 55.1-1245)
- **Repeat violations:** 30-day notice, no cure period required if substantially the same violation within 6 months

## Virginia-Specific Contract Rules

### Security Deposit (Residential)
- **Maximum:** 2 months' rent (Virginia Code 55.1-1226)
- **Return deadline:** 45 days after tenant vacates (Virginia Code 55.1-1226)
- **Itemized deductions required** with written notice
- **Interest:** Not required
- **Penalty:** If landlord fails to return within 45 days, tenant may recover full deposit plus reasonable attorney fees
- **Move-in inspection:** Landlord must provide written report of condition within 5 days of move-in; if not provided, landlord cannot withhold deposit for pre-existing conditions

### Contractor Licensing
Virginia requires contractor licensing through the Virginia Board for Contractors (DPOR):
- **Class A license:** Projects $120,000+ or 7,500+ sq ft
- **Class B license:** Projects $10,000-$120,000 or up to 7,500 sq ft
- **Class C license:** Projects $1,000-$10,000 (home improvement only)
- **Separate specialty licenses:** Electrical, plumbing, HVAC, gas fitting
- **Unlicensed work:** Class 1 misdemeanor; contracts may be unenforceable

### Mechanic's Lien (Virginia Code 43-1 et seq.)
- **Notice:** GC must provide memorandum of lien within 90 days of last work
- **Sub/supplier notice:** Must give written notice to owner within 30 days of first furnishing; lien filed within 90 days of last furnishing
- **Foreclosure deadline:** 6 months from lien filing (Virginia Code 43-17)
- **Virginia distinguishes between GC liens and sub/supplier liens** — different procedures for each

### Virginia Consumer Protection Act (Virginia Code 59.1-196 et seq.)
- Prohibits unfair or deceptive practices
- Private right of action for actual damages
- Attorney fees recoverable for individual actions (not class actions)
- Treble damages for willful violations (cap of $1,000 for individual actions under older case law; newer interpretations vary)
- Class actions have additional restrictions

### Source of Income Protection
Virginia prohibits housing discrimination based on source of income (including housing vouchers) under the Virginia Fair Housing Law — affects residential lease provisions.
