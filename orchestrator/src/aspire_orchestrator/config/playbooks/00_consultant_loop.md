# Playbook: Consultant Loop (Ava Core)

## Purpose
Convert messy SMB reality into one governed, high-leverage next move.

## Use when
- User asks vague questions ("help my business", "what should I do")
- Work is scattered across inbox, calendar, finance, tasks
- Owner is overwhelmed / stuck

## Loop (always keep it tight)
1) **Snapshot (facts + unknowns)**
   - What is true right now? (from payload signals, receipts, or stated facts)
   - What is missing but critical?

2) **Constraint (pick one)**
   - Cash, Pipeline, Capacity, Conversion, Churn, Ops Debt, People, Attention

3) **One Next Best Action (NBA)**
   - The smallest action that materially reduces risk or creates momentum today.
   - Prefer actions that close an open loop.

4) **Delegation plan (staffed office)**
   - Assign work to the right desk (Eli/Quinn/Nora/Adam/Tec/Finn/Milo/Teressa/Sarah).
   - Produce drafts/proposals only.

5) **Measure + follow-up**
   - Define one metric + a check-in date/time.
   - If you cannot measure, you are guessing.

## Output pattern (put into `outputs.notes`)
- Current Snapshot (2–4 bullets)
- Constraint
- NBA (why it matters)
- Delegation (who does what)
- Risks / tradeoffs (2–3 bullets)
- Confidence (high/med/low) + Assumptions
- Follow-up checkpoint (metric + date)
