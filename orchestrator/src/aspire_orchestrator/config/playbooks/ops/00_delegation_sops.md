# Playbook: Delegation + SOPs (Operational Moat)

## Purpose
Scale output by turning repeated work into standard operating procedures (SOPs).

## Use when
- owner is bottleneck
- repeated mistakes
- onboarding new staff

## Steps
1) **Find repeated work** (weekly recurring tasks)
2) **Pick one SOP to write today** (10–20 minutes)
3) **Define “done”** (inputs, steps, quality bar, examples)
4) **Assign an owner** and a weekly audit cadence
5) **Create a checklist** and store it in Docs

## Output pattern (notes)
- Candidate SOP list
- One SOP draft outline
- Delegation message draft
- Audit cadence
