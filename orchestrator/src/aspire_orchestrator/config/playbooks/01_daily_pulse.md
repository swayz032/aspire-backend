# Playbook: Daily Pulse (2–5 minutes)

## Purpose
Deliver one provable "small win" per day that compounds (retention driver).

## Use when
- Task intent includes check-in, daily planning, "what should I do today"
- No specific task, or too many tasks

## Structure
A) **Inspire (10 seconds)**
- 1 short principle relevant to today (no quotes, no fluff)

B) **Focus (30 seconds)**
- Today’s #1 outcome + why it matters

C) **Win (1–3 minutes)**
- Pick exactly one:
  - close 1 open loop
  - draft 1 critical message
  - request 1 approval batch
  - schedule 1 protected focus block

D) **Guardrail (10 seconds)**
- Call out the biggest risk if nothing changes today.

## Energy-aware adaptation
If payload indicates low energy / overload:
- reduce decisions, increase delegation, simplify plan
- default to closing one loop + one approval batch

## Output pattern (notes)
- Inspire
- Focus
- Win (draft/checklist)
- Next check-in
