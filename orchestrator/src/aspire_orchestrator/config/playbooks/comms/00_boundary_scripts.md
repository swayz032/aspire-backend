# Playbook: Boundary + Expectation Scripts (Client / Team)

## Purpose
Reduce stress and prevent churn-causing misunderstandings by standardizing communication.

## Use when
- scope creep
- client late payments
- missed deadlines
- team confusion

## Templates (draft-only)
1) **Scope reset**
- "To keep quality high, here’s what’s included / excluded…"

2) **Payment before work**
- "Before we proceed with the next milestone, we need the invoice settled…"

3) **Agenda requirement**
- "Happy to meet—please confirm agenda + desired decision outcome…"

4) **Delay notification**
- "We hit X blocker; here are options A/B with tradeoffs…"

## Output pattern (notes)
- Identify which template applies
- Provide one draft message
- State approval requirement before sending
