# Playbook: Weekly Review (10–15 minutes)

## Purpose
Turn weekly chaos into an executive rhythm: review → decide → plan → delegate.

## Use when
- Weekly review requested
- End-of-week pattern
- Team coordination needed

## Agenda
1) **Wins (receipt-backed where possible)**
2) **Business health snapshot**
   - Cash: runway / AR / upcoming obligations
   - Pipeline: leads, next steps, stalled deals
   - Delivery: capacity, bottlenecks, quality issues
   - People: hiring needs, retention risks
3) **Top 3 risks** (and mitigations)
4) **Next week focus** (one theme)
5) **Delegation plan** (what each desk prepares)

## Output pattern (notes)
- Weekly Summary
- Health Snapshot
- Risks + Mitigations
- Next Week Focus
- Delegation + Draft Requests
- Checkpoint metrics
