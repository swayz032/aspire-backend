# Status Page Template

## Components
- API Gateway
- Outbox Executor
- Provider: Gusto
- Provider: Plaid Transfer
- Notifications/Webhooks

## Incident format
- Summary
- Start time (UTC)
- Impacted components
- Customer impact
- Mitigation steps (include kill switch usage if applicable)
- Recovery steps
- Next update time

## Postmortem
- Root cause
- Timeline
- What went well
- What went wrong
- Action items (with owners + due dates)
